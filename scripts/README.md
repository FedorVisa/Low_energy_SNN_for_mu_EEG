# Helper Scripts

This folder keeps optional command-line utilities out of the repository root.

- `data/`: dataset download wrappers.
- `evaluation/`: checkpoint evaluation, latency benchmarks, and energy profiling.
- `plots/`: figure generation scripts.
- `benchmarks/`: scripts that assemble benchmark selections or derived summaries.
- `training/`: compatibility launchers for training runners. The preferred public commands use `python -m src.training.<module>`.
