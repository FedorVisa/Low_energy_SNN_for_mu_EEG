# Benchmark Artifacts

This directory is split by artifact type so the public repository stays readable:

- `results/`: curated JSON and CSV outputs used for summaries, README tables, and plot generation.
- `figures/`: generated plots and architecture diagrams grouped by benchmark family.
- `runs/`: local raw logs from grid searches and scratch runs. This directory is ignored by Git and is kept only for local inspection.

The most useful public files are usually under `results/full_eval`, `results/model_comparison`, and `results/model_efficiency`.
